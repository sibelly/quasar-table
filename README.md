## Usage
Run this project using this template:

``` bash
# install Quasar CLI if you don't have it already
$ npm install -g quasar-cli

# checkout/clone the project
$ git clone https://github.com/username/quasar-table.git

$ cd quasar-table
$ npm install
$ quasar dev
```
